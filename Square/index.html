The Basic sample for the Python platform has the same requirements as the Python client library. You will also require the Bottle and Beaker Python packages, as the sample is implemented as a simple web services application in a Bottle framework using Beaker to manage session state data. To install. First, download the latest version of the Python sample repository. Then, unzip the contents into a temporary folder. To run the sample. From within the Basic sample subdirectory, run the basic.py script to start the Bottle web-service application. Then, in a web-browser, navigate to http://localhost:8080/start.
